## 使用方法：
python 3.7.9
打开你的ff14，食物+魔匠，窗口模式，蹲下，打开轮椅。
然后根据提示在游戏中释放技能，同时在轮椅里点击对应的色球和技能，失败按“失败10耐”按钮
内静8层+效率2450进入收尾阶段，有五种手法，见nga链接，如果提示没有的话是前面炸的多了不能直接收尾，考虑注视加工钓绿球+仓促苟
